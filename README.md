# Platform E-Learning / Streaming Course

Aplikasi web e-learning berbasis **CodeIgniter 4** (PHP): pendaftaran kursus, konten pembelajaran, dashboard pengguna, dan panel admin.




| Periode        | Keterangan                                                                                                                                                                    |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Semester 4** | Proyek ini dikerjakan pada **Senin, 2 Juni 2025** untuk memenuhi **UAS (Ujian Akhir Semester)**.                                                                              |
| **Semester 6** | Dikembangkan kembali untuk menambah fitur **pembayaran (payment)** dan **RBAC (role-based access control)** — juga digunakan untuk memenuhi **proyek UAS** semester tersebut. |
